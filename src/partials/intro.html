Всё гениальное - просто. Нагромождение сложностей снижает продуктивность. Чем проще код, тем легче его читать и тестировать. Моя работа, как разработчика состоит в том, чтобы писать код без ошибок, работающий, как швейцарские часы.

Complexity is the enemy of productivity. Simple code with fewer moving parts is easier to read and test. As a developer my job is to write code without bugs that works like a swiss clock.
